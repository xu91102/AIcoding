# 学习状态命令

查看 AI 助手当前学习到的行为模式、置信度和优化建议。

## 使用方法

### Claude 用户
```markdown
请执行学习状态检查：

按照 .ai-assistant/commands/learning-status.md 的格式，
调用 .ai-assistant/skills/adaptive-learning/SKILL.md 的分析能力，
显示当前学习到的所有行为模式。
```

### ChatGPT 用户
```markdown
请显示 AI 助手学习状态：

## 分析范围
- 代码风格偏好
- 工作流程模式
- 架构设计习惯
- 调试方法偏好

## 输出格式
按照学习状态报告格式显示结果
```

### 通用平台
```bash
# 如果有脚本支持
python .ai-assistant/learning/scripts/status.py

# 手动检查
ls .ai-assistant/learning/patterns/
cat .ai-assistant/learning/patterns/*/confidence-*.md
```

## 命令参数

### 详细模式
```bash
/learning-status --detailed
# 显示每个模式的详细证据和学习历史
```

### 领域过滤
```bash
/learning-status --domain=code-style
# 只显示代码风格相关的学习模式
```

### 置信度过滤
```bash
/learning-status --min-confidence=0.8
# 只显示高置信度的模式
```

### 导出格式
```bash
/learning-status --export=json
# 以 JSON 格式导出学习数据
```
